Cool Air Services - Demo one-page site
-------------------------------------

Files:
- index.html
- css/style.css

How to use:
1. Unzip and open index.html in your browser to view locally.
2. To embed your Typeform chatbot, replace the iframe src in the Chat section:
   <iframe src="https://form.typeform.com/to/XXXX" ...></iframe>
3. To deploy:
   - Drag & drop the unzipped folder at https://app.netlify.com/drop
   - or push to GitHub and use GitHub Pages / Vercel.
